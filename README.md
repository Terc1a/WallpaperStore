# Wallpaper Store

Wallpaper Store — это веб-приложение на Django для хранения, просмотра и управления коллекцией обоев.

## Возможности
- Загрузка и хранение изображений обоев
- Категоризация и тегирование
- Просмотр детальной информации о каждом изображении
- Удобный интерфейс для поиска и фильтрации

## Структура проекта
- `wall_store/` — основной Django-проект
- `walls/` — приложение для работы с обоями
- `media/` — директория для хранения загруженных изображений

## Быстрый старт
1. Клонируйте репозиторий:
   ```sh
   git clone https://github.com/Terc1a/WallpaperStore.git
   ```
2. Установите зависимости:
   ```sh
   pip install -r requirements.txt
   ```
3. Примените миграции:
   ```sh
   python wall_store/manage.py migrate
   ```
4. Запустите сервер:
   ```sh
   python wall_store/manage.py runserver
   ```
5. Откройте [http://127.0.0.1:8000/](http://127.0.0.1:8000/) в браузере.
